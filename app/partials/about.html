About

This is the Cluster Manager used to configure the whole cluster ox, it contains several tools to manage and controls the cluster.

The Modules tool lets the user change or update all the Cluster components such as connectors libraries, dictionaries, etc.

The Registry tool allows to update a private registry that contains all the images (OS, APPs) to be used inside the cluster, the Units Tool can be used to create and send units that will create and run a complete container with everything inside (OS, APPs and configuration files).

In addition, it is possible to get metrics regarding the Cluster health as well as info about the System
